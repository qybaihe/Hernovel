# 角色卡

## 男主：周临
- 身份：大一新生
- 外在状态：普通、存在感低、会察言观色
- 核心性格：能忍、敏感、退让、自尊心低但不是真的废
- 核心缺点：怕给别人添麻烦，长期自我压低
- 核心成长：学会不再一味退让，敢于主动做决定
- 情感吸引点：他不是天降强者，而是一步步被逼着站起来

## 女主：顾沉霜
- 身份：大三、学校风云人物
- 外在状态：强势、克制、压场、执行力极强
- 核心性格：冷静、掌控欲强、边界感强
- 情感危险点：她的靠近并不温柔，而是带压迫感、审视感和控制感
- 内在裂缝：她对关系与信任高度敏感，不轻易交出自己
- 关系弧线：从审视男主，到失控地在意男主

## 关系定义
- 前期：压迫 / 审视 / 被盯上
- 中期：控制 / 依赖 / 试探
- 后期：失控 / 心动 / 对等
